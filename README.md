# Weblico

The platform which will create my personal dynamic webapp

| Dependency | Version |
|:-----------|:--------|
| Ruby       | 2.6.5   |
| Rails      | 6.0.2.1 |
| Postgresql | 1.2.2   |

## Getting Started

1. Clone the project

  `git clone git@github.com:EGiataganas/weblico.git`

2. Install the project's dependencies using bundler

  `bundle install`

## Server

  `rails s`
